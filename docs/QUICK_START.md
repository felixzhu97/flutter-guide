# 快速开始指南

## 🚀 5 分钟快速体验

### 1. 环境检查

```bash
# 检查Flutter版本
flutter --version

# 检查设备连接
flutter devices

# 检查环境配置
flutter doctor
```

### 2. 运行项目

```bash
# 安装依赖
flutter pub get

# 运行应用
flutter run
```

### 3. 核心功能体验

#### 🔢 计数器功能

- 点击主页的"计数器"卡片
- 使用 + 和 - 按钮进行计数
- 点击重置按钮清零

#### 📝 待办事项

- 点击主页的"待办事项"卡片
- 添加新的待办事项
- 标记任务完成状态
- 删除不需要的任务

#### 🌐 网络请求

- 点击主页的"网络请求"卡片
- 查看从 API 获取的用户列表
- 体验加载状态和错误处理

#### 🎨 主题切换

- 点击主页的"主题"卡片
- 或点击右上角的设置按钮
- 切换浅色/深色/系统主题

#### 🌍 语言切换

- 点击主页右下角的语言按钮
- 或进入设置页面
- 切换中文/英文界面

### 常见问题

#### 1. 依赖安装失败

```bash
# 清理缓存
flutter clean
flutter pub cache repair
flutter pub get
```

#### 2. 设备连接问题

```bash
# 重启ADB服务
adb kill-server
adb start-server
```

#### 3. 编译错误

```bash
# 清理项目
flutter clean
flutter pub get
flutter run
```
